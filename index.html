Interactive Simulations & Games

Things that move in a browser.

A collection of interactive simulations, games, and small apps — each built from scratch with no engine, no framework, and (almost) no dependencies. Physics, chaos, gravity, fluids, and play, all running live.

None of these began as my own idea. Each one recreates an existing project or concept that caught my eye — and curiosity about how it actually works led me to rebuild it my own way, and make it a little different and better as I went.

Hand-built with vanilla JavaScript, Canvas, WebGL, and a little Three.js. Open each card to run it live, or follow Code to the source.
