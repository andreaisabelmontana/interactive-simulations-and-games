# Interactive Simulations & Games

A hub for a collection of from-scratch, browser-based **interactive simulations, games, and small apps** — physics, chaos, gravity, fluids, and arcade play, each built with vanilla JavaScript, Canvas, WebGL, or a little Three.js.

**▶ Live:** https://andreaisabelmontana.github.io/interactive-simulations-and-games/

> **Not an original idea.** These recreate the concepts of existing projects — I didn't invent them. I rebuilt each one from scratch, my own way, out of curiosity about how it actually works (and tried to make it a little better along the way).

## In the collection

**Simulations** — [Solar System Simulation](https://andreaisabelmontana.github.io/solar-system-simulation/) · [Black Hole Simulation](https://andreaisabelmontana.github.io/black-hole-simulation/) · [Fluid Simulation](https://andreaisabelmontana.github.io/fluid-simulation/) · [Ant Colony Simulation](https://andreaisabelmontana.github.io/ant-colony-simulation/) · [Boids Flocking](https://andreaisabelmontana.github.io/boids-flocking/) · [Strange Attractors](https://andreaisabelmontana.github.io/strange-attractors/) · [Fractal Explorer](https://andreaisabelmontana.github.io/fractal-explorer/)

**Games** — [Chess Dodge](https://andreaisabelmontana.github.io/chess-dodge-game/) · [Tilt Maze](https://andreaisabelmontana.github.io/tilt-maze-game/) · [Pong](https://andreaisabelmontana.github.io/pong/) · [Snake](https://andreaisabelmontana.github.io/snake/) · [Echolocation Maze](https://andreaisabelmontana.github.io/echolocation-maze/)

**Apps & Data** — [Rental Finder](https://andreaisabelmontana.github.io/rental-finder/) · [Bank Marketing ML](https://andreaisabelmontana.github.io/bank-marketing-ml/) · [Chess Dodge Server](https://github.com/andreaisabelmontana/chess-dodge-server)

## Tech

Static site — `index.html` + `data.js` (the catalogue) + `app.js` (renders the cards and a quiet boids backdrop). No build step, no dependencies.
